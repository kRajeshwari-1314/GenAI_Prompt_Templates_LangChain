# GenAI Prompt Templates using LangChain 🚀

## 📌 Overview
This project demonstrates how to build dynamic and reusable prompt systems using LangChain.  
Instead of hardcoding prompts, we use PromptTemplate and ChatPromptTemplate to create flexible and scalable prompt pipelines.

---

## 🎯 Objective
- Replace hardcoded prompts with reusable templates  
- Build dynamic prompt generation systems  
- Implement structured prompting using LangChain  
- Design a mini prompt engine with validation and multiple styles  

---

## 🛠️ Technologies Used
- Python 🐍  
- LangChain  
- Jupyter Notebook / Google Colab
